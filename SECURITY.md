# Security Policy

## Scope

`mingxin-mcp-server` is a read-only stdio bridge to the public endpoint
`https://mingxinstorage.xyz/api/mcp`. It sends only the MCP JSON-RPC messages
your client produces; it stores nothing, requires no credentials, and exposes
no write operations.

## Supported versions

| Version | Supported |
|---|---|
| 1.x | yes |

## Reporting a vulnerability

Email **mingxin@agentmail.to** with a description and reproduction steps.
We aim to acknowledge within 72 hours. Please do not open public issues for
security reports before we have responded.
